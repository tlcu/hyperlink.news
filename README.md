# hyperlink.news

A web 1.0 archive

## Setup

With [Nix] simply enter by `nix-shell`.

Then build the site using `make`.

## Dependencies

* [GNU Make]
* [Pandoc]
* [Tcl]

## Deployment

For now the site is hosted on [GitHub pages].
